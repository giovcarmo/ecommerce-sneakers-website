# E-Commerce Website - Sneakers

Neste desafio eu quis provar da experiência de ser um Dev Junior, e construir toda uma frontend page. Era necessário criar apenas uma página de um produto, ver as fotos do produto, seu nome, descrição, preço e desconto, podendo ampliar e passar as fotos além de adiciona-lo no carrinho. No meu projeto, eu fui além e construí o site todo, então além do produto indicado pelo frontend mentor, eu adicionei mais 7 todos estruturados com as informações necessárias. O usuário pode navegar pela home page e ver todos os produtos, ou navegar pelas abas women e men para separa-los. Na sessão About deixei uma breve informação sobre mim, em Contact, minhas informações de contato. Construí também a parte do Carrinho, onde além do usuário poder adicionar o produto, você pode clicar para seguir com a compra, você será convidado a preencher os dados para envio do produto. Também adicionei o botão para troca de tema, light e dark(aprendido no projeto passado do curso), para experienciar melhor sua navegação. E tudo isso feito com um design responsivo.
